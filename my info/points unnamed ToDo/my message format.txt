информация передается строчками

toWhom\table action\table_name/table_action/message_numb/node_numb/game_action/
type=server information\?



tw\ta\tn\
tw\si\gsi

tw= to whom
ta= table action
si= server information
gsi= get server information


toWhom/

toWhom =
кому это сообщение? Серверу, оппоненту,...

table_name = 
название игровой доски где должно произойти событие

table_action =
s - отправка своего сообщения (N)
d - удаление всех сообщений начиная с N (включительно)
r - запрос на переотправку сообщения N
R - на переотправку всех сообщений начиная с N

message_numb = 
номер текущего сообщения в этой доске.
(Использовать для синхронизации и устранения ошибок)
"%" - игнорировать синхронизацию.

node_numb = 
вершина, в которой будет происходить действие
"!" - в текущей вершине
"+" - в новой вершине, созданной от текущей
"&" и дальше номер - в новой вершине, созданной от..
Обычные значения: "!" в случае комментариев и "+" в случае новых ходов.

game_action = действие
. - просто перейти в эту вершину
& - добавить в вершину текст sgf
$ - очистить эту вершину
# - удалить вершину, дочерей повыдавать замуж. Если вершина - корневая, то полностью очистить дерево.
0,9,M,.. - удалить свойства, по указанным индексам (индексы через знак запятую)
